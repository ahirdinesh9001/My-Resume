This project is a single-page resume website built using only HTML.
It is designed for beginners and follows a clean, semantic structure without using any CSS or JavaScript.

The goal of this assignment is to demonstrate:

Proper HTML document structure

Use of semantic HTML tags

Lists, tables, and sections

A real-world resume layout

Features

Single HTML file

No CSS or JavaScript (as per assignment requirement)

Semantic HTML elements

Easy to read and modify

Beginner-friendly structure

Included sections:

Header / Name

About Me

Skills

Work Experience (Table)

Projects

Education (Table)

Contact Details
